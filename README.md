### Plugin Template

Papermc plugin template for minecraft version 1.21.x with implements.
 - PaperMC
 - Lamp Command Framework
 - MiniMessage

Getting Started
 - [Wiki](https://github.com/d4nilpzz/plugin-template/wiki)
 - [First Steps](#first-steps)

### First Steps
Clone the repository and run the gradlew build task.

Adjust the `gradle.properties` file with your plugin information.

Then you can run the server with `./gradlew runServer` as debuger.

> [!IMPORTANT]
> 
> Use debug mode for development to use hotswap.
> 
> You need to have java 17 & 21 installed.
